Play both normal mode and the including reskins challenge mode

If you get an error code when clicking play, try refreshing the page (this will hopefully be fixed later either by github or by Godot)


FAQ

- What does it mean by "slot"?

TF2 Weapons can be in a few character items slots: Primary, Secondary, melee, building, PDA 1, and PDA 2


- What weapons are in the building slot?

The buliding slot only includes the spy's sapper items


- I got all green boxes but didn't win?

Although rare, there are a few weapons that share all 5 characteristics. For example, the Buffalo Steak Sandvich and Dalokhos Bar are both heavy secondaries released in 2010 which are not reskins

I used Godot Game Engine!
